How to export a PNG

  1. Open any asset above. Make sure the dev server is running on the same origin.
  2. Right-click the banner → Inspect (opens DevTools).
  3. In the Elements panel, right-click the <div class="canvas"> element.
  4. Choose Capture node screenshot. PNG downloads at exact pixel dimensions.
  5. Alternative: use a full-page screenshot extension if your browser doesn't support node capture.
